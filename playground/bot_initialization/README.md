# Bot Initialization

Ссылка на бота: https://t.me/employee_mse_bot

## Запуск
1. **Конфигурация:** Файл `.env` с токеном (`BOT_TOKEN`) необходимо запросить у админа бота.
2. **Установка зависимостей:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Запуск бота:**
   ```bash
   python main.py
   ```
