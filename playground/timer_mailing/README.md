# Timer mailing

1. **Конфигурация:** Токен (`MAILING_INTERVAL_SECONDS`) в `.env` файле можно использовать для инициализации периода рассылки, по умолчанию рассылка происходит раз в час.
2. **Зависимости:** Необходимо убедится, что все зависимости из файла `requirements.txt` были установлены, добавить недостающие в случае отсутствия.
3. **Интеграция:** Необходимо импортировать файл `mailing.py`, вызвать функции `register_mailing` и `start_scheduler` во время инициализации бота.
4. **Функционал:**
- Подписка на рассылку производится с помощью команды `/subscribe`.
- Отписка производится с помощью команды `/unsubscribe`.
- Рассылка отправляется всем подписанным пользователям.
