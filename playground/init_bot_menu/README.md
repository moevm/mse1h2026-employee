#Init bot menu

## Описание
В рамках задания реализованы
1. приветственное меню бота(`welcome_menu`)
2. меню сотрудника(`employee_menu`)

**Зависимости:** Необходимо убедится, что все зависимости из файла `requirements.txt` были установлены, добавить недостающие в случае отсутствия.
**Интеграция:** Необходимо импортировать файлы `welcome_menu.py` и `employee_menu.py`.
Вместо обработчика команды старт вызвать функции `register_welcome_menu(dp)` и `register_employee_menu(dp)`
